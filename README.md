# HoliSource — The Delegation 2 · Summer 3D

Visualisation interactive du graphe de connaissances HoliSource en 3D.

**[➜ Ouvrir le graphe](https://auralinktherapeute.github.io/holisource-graph/)**

## Fonctionnalités

- 597 nœuds · 920 arêtes · 21 communautés
- Bloom post-processing (UnrealBloomPass)
- Champ d'étoiles animé
- Particules en courbes de Bézier avec trails
- Caméra fluide avec lerp
- Panneau Admin (touche `A` — mot de passe : `holisource`)
  - Analytiques du graphe
  - Export CSV nœuds / arêtes
  - Annotation et masquage de nœuds
  - Visibilité par communauté

## Stack

Three.js r128 · HTML/CSS/JS · Aucune dépendance serveur
